Array: length

Like strings, arrays have the property length. Here it returns the number of elements in the array:
var languages = [];
var l0 = languages.length;
languages[0] = "C";
var l1 = languages.length;
languages[1] = "C++";
var l2 = languages.length;
The empty array contains no elements. l0 is 0. Now we fill the array step by step. l1 is 1 and l2 is 2.

Fahrenheit

Die folgende Aufgabe ist dem freeCodeCamp entnommen.

Aufgabe

Schreibe eine Funktion toFahrenheit, die eine Temperaturangabe von Grad Celsius in Grad Fahrenheit umrechnet.

Tipp: Ist C die Temperatur in Grad Celsius und F die Temperatur in Grad Fahrenheit, so gilt: F = 1.8 * C + 32.

Zum Beispiel: toFahrenheit(0) sollte 32 ergeben.
+ Tipp 
var toFahrenheit = function(celsius) {
  return ...
};
+ Lösung 
var toFahrenheit = function(celsius) {
  return 1.8 * celsius + 32;
};

Der Browser-Speicher localStorage ist nicht verfügbar. Entweder unterstützt dein Browser ihn nicht oder du hast ihn deaktiviert oder er ist voll. Ohne localStorage werden deine Lösungen nicht gespeichert.

Array-Elemente lesen

Die Elemente eines Arrays werden mit einem Index beginnend mit 0 ausgelesen. 
var sprachen = ['C', 'C++', 'Java', 'JavaScript'];
var c = sprachen[0];
var cPlusPlus = sprachen[1];
var java = sprachen[2];
var javaScript = sprachen[3];

Aufgabe

Schreibe eine Funktion getFirstElement, die ein Array entgegennimmt und die das erste Element des Arrays zurückgibt.

Beispiel: getFirstElement([1, 2]) sollte 1 zurückgeben.

Strikte Ungleichtheit

Mit !== vergleicht man zwei Werte auf strikte Ungleichheit. 
let v1 = 'lauf' !== 'Lauf';
let v2 = 10 !== '10';
Beide Vergleiche ergeben true. Der erste, weil sich die beiden Strings in der Groß- und Kleinschreibung unterscheiden. Der zweite, weil sich die beiden Werte von Typ her unterscheiden.

Aufgabe

Schreibe eine Funktion unequal, die 3 Werte auf strikte Ungleichheit überprüft. Die Funktion soll nur dann true ergeben, wenn alle drei Parameter unterschiedlich sind.

Beispiel: unequal(1, 2, 3) soll true liefern, unequal(1, 1, 2) soll false ergeben.
+ Tipp 
function unequal(a, b, c) {
  return a !== b && ...
}
+ Lösung 
function unequal(a, b, c) {
  return a !== b && a !== c && b !== c;
}

Der Browser-Speicher localStorage ist nicht verfügbar. Entweder unterstützt dein Browser ihn nicht oder du hast ihn deaktiviert oder er ist voll. Ohne localStorage werden deine Lösungen nicht gespeichert.

String: length

Strings besitzen die Eigenschaft length. Diese Eigenschaft liefert die Anzahl der Zeichen eines Strings. Dazu muss man dem String oder der Variablen, die den String enthält, .length anhängen: 
let anzahl = 'Franz'.length;
'Franz' hat 5 Zeichen. Also erhält anzahl den Wert 5. 
let stadt = 'Prag';
anzahl = stadt.length;
'Prag' hat 4 Zeichen. anzahl hat jetzt den Wert 4. 
anzahl = ''.length;
Der Leerstring besitzt keine Zeichen. Der Wert von anzahl ist jetzt 0.

Aufgabe

Schreibe eine Funktion laenge, die bei Übergabe eines Strings die Anzahl der Zeichen dieses Strings zurückgibt.

Beispiel: laenge('Franz') sollte 5 zurückgeben.
+ Tipp 
let laenge = function(s) {
...
};
+ Lösung 
let laenge = function(s) {
  return s.length;
};

String: substr()

Die Methode substr extrahiert aus einen String einen Teilstring: 
let see = 'see and stop'.substr(0, 3);
let and = 'see and stop'.substr(4, 3);
let stop = 'see and stop'.substr(8);
Der erste Parameter gibt an, von welcher Stelle an der Substring extrahiert werden soll. Der zweite Parameter gibt an, wieviel Zeichen ab der festgelegten Stelle extrahiert werden sollen. Ist der zweite Paramter nicht angegeben, so wird immer bis zum Ende des ursprüglichen Strings extrahiert.

Aufgabe

Schreibe eine Funktion firstWord, die einen String entgegennimmt und das erste Wort in diesem String zurückgibt. Das erste Wort sind dabei alle Zeichen bis zum ersten Leerzeichen.

Beispiel: firstWord('Wo ist hier') sollte 'Wo' zurückgeben.
+ Tipp 
function firstWord(s) {
// Bestimme mit indexOf
// die Position des
// ersten Leerzeichens.
// Verwende dann die
// substr-Methode.
}
+ Lösung 
function firstWord(s) {
  let firstBlank = s.indexOf(' ');
  return s.substr(0, firstBlank);
}

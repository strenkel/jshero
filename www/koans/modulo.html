Modulo

Ein weiterer arithmetischer Operator ist Modulo. Er berechnet den Rest einer Division und wird mit dem Prozentzeichen % notiert.
var x = 7 % 2;
var y = 20 % 3;
7 geteilt durch 2 ergibt 3 Rest 1. x ist also 1.
20 geteilt durch 3 ergibt 6 Rest 2. y ist also 2.

Schreibe eine Funktion einer, die eine natürliche Zahl entgegenimmt und die Ziffer der Einer-Stelle zurückgibt. einer(2674) sollte 4 zurückgeben.
+ Tipp 
var einer = function(n) {
// Benutze Modulo 10.  
};

var einer = function(n) {

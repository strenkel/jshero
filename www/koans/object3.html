Eigenschaften auslesen

Auf Objekt-Eigenschaften kann man mit der Punkt-Notation zugreifen: 
var person = {
  name: 'Constanze Kurz',
  geburtstag: new Date(1974, 2, 2),
  geburtsort: 'Ost-Berlin'
};

var pName = person.name;
var pGeburtstag = person.geburtstag;
var pGeburtsort = person.geburtsort;
Die drei Eigenschaften des Objekts person werden in drei Variablen gespeichert.

Aufgabe

Schreibe eine Funktion distance, die den Abstand eines Punkts zum Nullpunkt berechnet. Hier und im Folgenden ist ein Punkt ein Objekt mit den Eigenschaften x und y. distance({x: 3, y: 4}) sollte 5 zurückgeben.
+ Tipp
Der Abstand eines Punkts mit den Koordinaten x und y zum Nullpunkt ist √(x² + y²).
+ Lösung 
var distance = function(point) {
  var xSquare = Math.pow(point.x, 2);
  var ySquare = Math.pow(point.y, 2);
  return Math.sqrt(xSquare + ySquare);
};

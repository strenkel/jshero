Array: length

Die Eigenschaft length hatten wird schon bei Strings kennengelernt. Dort gab sie die Anzahl der Zeichen eines Strings zurück. Bei Arrays gibt sie die Anzahl der Elemente zurück. 
var sprachen = [];
var l0 = sprachen.length;
sprachen[0] = 'C';
var l1 = sprachen.length;
sprachen[1] = 'C++';
var l2 = sprachen.length;
Das leere Array zu Anfang enthält keine Elemente. l0 ist also 0. Nun füllen wir das Array nach und nach. l1 und l2 haben dann die Werte 1 und 2. Beachte: Die Länge eines Arrays ist immer um 1 größer als der höchste Index des Arrays.

Aufgabe

Schreibe eine Funktion getLastElement, die ein Array entgegennimmt und die das letzte Element des Arrays zurückgibt.

Beispiel: getLastElement([1, 2]) sollte 2 zurückgeben.
+ Tipp
Bestimme den Index des letzten Elements mit der Array-Eigenschaft length. Beachte, dass das erste Element im Array den Index 0 besitzt.
+ Lösung 
var getLastElement = function(anArray) {
  var lastIndex = anArray.length - 1;
  return anArray[lastIndex];
};
